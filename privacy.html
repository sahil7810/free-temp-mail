Privacy Policy

Overview

QuickTemp Mail is a beginner-friendly static website project for temporary email use. This placeholder version does not connect to a backend, database, Firebase, or any paid service.

Information We Collect

This static version does not collect personal information through forms, accounts, or a database.

Temporary Email Usage

Future versions may use the public Mail.tm API to create temporary inboxes. Temporary email should be used for privacy-friendly testing and spam protection, not for important accounts.

Local Storage

The website may use your browser local storage only to remember your light or dark theme choice.

Third-Party Services

This placeholder version does not load external UI libraries or API keys. If Mail.tm API support is added later, messages may be processed according to Mail.tm's own service behavior.
