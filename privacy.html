Overview

QuickTemp Mail is a static website that connects directly to the public Mail.tm API from your browser. It does not use a backend, database, Firebase, paid service, or API key.

Information Stored in Your Browser

To keep your temporary inbox available after refresh, this website stores the current temporary email address, generated password, access token, account id, last refresh time, and theme choice in your browser localStorage.

Mail.tm API

When you generate an email or refresh your inbox, your browser sends requests to Mail.tm. Messages are retrieved from Mail.tm and displayed as safe plain text where possible.

Email Content

Temporary inbox messages are not stored in this website's own database. Do not use temporary email for important accounts, private recovery emails, banking, school, government, or long-term access.

Third-Party Services

This website uses Mail.tm for temporary email functionality. It does not load external UI libraries.
