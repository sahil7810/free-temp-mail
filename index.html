Questions

FAQ

Quick answers about temporary email, disposable email, privacy, and safe use.

What is a temporary email?

A temporary email is a short-term inbox you can use when you do not want to share your main email address.

Is temp mail useful for spam protection?

Yes. A temp mail inbox can help reduce unwanted messages in your real inbox when you only need a short-term address.

Is QuickTemp Mail fully anonymous?

No. This project does not claim full anonymity. Use it for privacy-friendly temporary email needs and basic testing.

Can I use disposable email for important accounts?

No. Temporary inboxes are not recommended for banking, school, government, recovery emails, or anything important.

Does QuickTemp Mail need an API key?

No. The website uses the public Mail.tm API from the browser and does not require an API key.
