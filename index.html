Questions

FAQ

Simple answers before you use a temporary inbox.

What is a temporary email?

A temporary email is a short-term inbox you can use when you do not want to share your main email address.

Why should I use QuickTemp Mail?

It can help reduce spam in your real inbox and is useful for simple testing or sign-up experiments.

Is this website fully anonymous?

No. This project does not claim full anonymity. Use it mainly for privacy-friendly temporary email needs and basic testing.

Can I use it for important accounts?

No. Temporary inboxes are not recommended for banking, school, government, recovery emails, or anything important.

Does this version receive real emails?

Not yet. The JavaScript file currently has placeholder functions only. Real inbox support can be added later using the public Mail.tm API.
